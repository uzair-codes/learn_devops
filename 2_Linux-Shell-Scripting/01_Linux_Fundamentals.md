# Section 1: Linux (Ubuntu), Why Linux, Architecture, Distributions

## 🐧 What is Linux?

* Linux is an **open-source operating system**
* It is used to manage **hardware and software resources**
* It is widely used in:

  * Servers
  * Cloud (AWS, Azure, GCP)
  * DevOps tools (Docker, Kubernetes)

👉 In DevOps, **Linux is the most commonly used OS**

---

## ❓ Why Linux is Used in DevOps?

### Key Reasons

* **Free & Open Source**
* **Stable & Reliable**
* **Secure**
* **Lightweight (uses less resources)**
* **Powerful Command Line (CLI)**
* **Automation Friendly (Shell Scripting)**
* **Best Support for DevOps Tools**

---

## 🧱 Linux Architecture (Simple View)

Linux works in layers:

User → Shell → Kernel → Hardware

### 1. Hardware

* Physical components
* CPU, RAM, Disk, Network

### 2. Kernel (Core of Linux)

* The **brain of the OS**
* Directly interacts with hardware

#### Responsibilities

* Process management
* Memory management
* Device control
* File system management

### 3. Shell

* Interface between **user and kernel**
* Accepts commands and sends them to kernel

#### Examples

* bash (most common)
* sh
* zsh

### 4. User Space

* Where users and applications run
* Includes:

  * Terminal
  * Applications
  * Scripts

---

## 📦 Linux Distributions (Distros)

A Linux distribution = **Linux kernel + tools + software**

### Popular Linux Distributions

#### 🔹 Ubuntu

* Beginner-friendly
* Most popular in DevOps
* Based on Debian

#### 🔹 CentOS / Rocky Linux

* Used in enterprise environments
* Stable and secure

#### 🔹 Red Hat Enterprise Linux (RHEL)

* Paid version
* Enterprise support

#### 🔹 Amazon Linux

* Optimized for AWS

#### 🔹 Debian

* Very stable
* Base for Ubuntu

### 💡 Simple Understanding

* Kernel = Core
* Distribution = Full OS (with tools)

---

## 🔹 Quick Revision Summary

* Linux is an **open-source OS** widely used in DevOps
* Key benefits: **secure, stable, lightweight, automation-friendly**
* Architecture: **User → Shell → Kernel → Hardware**
* Kernel = core of OS
* Shell = command interface
* Popular distros: **Ubuntu, RHEL, CentOS, Amazon Linux**

---

## 🔹 Interview Key Points

* Linux is preferred because of **stability and CLI power**
* Kernel manages **hardware and system resources**
* Shell acts as a **bridge between user and kernel**
* Ubuntu is widely used in **cloud and DevOps environments**
* Distribution = kernel + tools + packages
